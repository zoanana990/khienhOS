## Environment Setup
Operating System: Ubuntu Linux 20.04

Install some repositories:
```shell
sudo apt update
sudo apt install nasm
sudo apt install qemu-system-x86
```
Check the `nasm` is installed:
```
nasm -v
```
Expected:
```
NASM version 2.14.02
```
Check the `qemu` is installed:
```
qemu-system-x86_64
```
Expected:

A QEMU window
